## [1.0.0] - 2026-04-14

### Added
- 🎉 Initial release of Language Exchange Platform
- 🔐 User authentication system with JWT
- 👥 Friend request and management system
- 💬 Real-time chat functionality with Stream.io
- 📹 One-on-one video calling with screen sharing
- 🌍 Language profile setup (native and learning languages)
- 🎨 32 beautiful themes with dark/light mode
- 📱 Fully responsive design for all devices
- 🔔 Real-time notifications for friend requests
- 📁 Avatar upload and profile customization
- 🚀 Onboarding flow for new users
- 🔒 Secure password hashing with bcrypt
- 🍪 HTTP-only cookie-based session management
- ⚡ Fast and optimized frontend with Vite
- 🎯 Modern UI with TailwindCSS and DaisyUI

### Backend Features
- Express.js REST API
- MongoDB database integration
- Multer for file uploads
- Socket.io for real-time features
- Stream Chat SDK integration
- CORS and security middleware
- Environment-based configuration

### Frontend Features
- React 19 with modern hooks
- React Router for navigation
- TanStack Query for data fetching
- Zustand for state management
- Stream Video SDK integration
- Lucide React icons
- Toast notifications
- Custom hooks for authentication

### Security
- JWT token authentication
- Password strength validation
- HTTP-only cookies
- CORS configuration
- Input validation and sanitization
- Secure file upload handling

### Documentation
- Comprehensive README with setup instructions
- Architecture diagrams
- API documentation

- Contributing guidelines
