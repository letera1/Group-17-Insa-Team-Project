Language Learning Platform

This is a work-in-progress project for building a language learning platform.

##  Project Goals

- Help users learn new languages interactively  
- Include vocabulary, grammar, listening, and speaking practice  
- Track user progress and achievements  


## Planned Tech Stack

- **Frontend**: React + CSS  
- **Backend**: Node.js + Express  
- **Database**: MySQL
